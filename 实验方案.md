## 角色设计和 Prompt

已经确定 Teacher 和 Student 两个角色。

> 理由：本质区分为有知识库和无知识库两种角色，有知识库的角色行为上可以是老师，助教，专家的集合体。

超参数：课时，教学大纲，用 system 的角色带入。

自主模拟遗忘曲线，曲线为指数函数。

THULAC 分词器，对上下文中的 token 替换抹除掉来模拟遗忘。

学生的学习成果评判标准：首先让老师自己去做一遍试题，然后再让学生去做一遍试题，让老师去评价和打分。
